# introduction‑app

一款使用 Flutter 構建的「自我介紹」應用程式，適合作為入門範例或個人／團隊展示用途。

## 特性
- 使用 Flutter 支援多平台（Android、iOS、Web、Desktop）  
- 簡潔直觀的 UI 設計，方便展示個人資訊、技能、經歷  
- 支援圖片、文字內容與版面佈局調整  
- 易於擴展：可新增頁面、模塊、主題風格  

## 快速開始 🚀
以下說明以你本機開發為例。

### 環境需求
- Flutter SDK （建議版本：最新穩定版）  
- 已安裝相應平台的開發工具（如 Android Studio + Android SDK、Xcode）  
- Git
